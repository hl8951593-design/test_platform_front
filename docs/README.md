# 文档索引

| 文档 | 说明 |
| --- | --- |
| `frontend-feature-logic.md` | 前端当前功能逻辑，包括认证刷新、环境配置、测试计划、接口用例、多环境绑定、调试响应和测试覆盖 |
| `frontend-performance.md` | 前端性能、请求竞态、列表状态和表单维护约定 |
| `technical-architecture.md` | 项目整体技术架构、业务模块、权限和数据关系 |
| `test-plan-architecture.md` | 测试计划模块前端架构、数据模型、本地持久化、能力边界和后端接入建议 |
| `scenario-composer-architecture.md` | 场景组合模块前端架构、步骤与数据集模型、模拟运行规则和后端接入建议 |
| `api_environment_configs.md` | 环境配置相关后端接口文档 |
| `visual-flow-architecture.md` | 可视化编排模块前端交互、取值关联与执行模型 |
| `../api_docs/api_visual_flows.md` | 可视化编排后端接口、数据结构、校验、执行器和数据库契约 |
| `../api_docs/visual_flow_schema.json` | 可视化编排 FlowDefinition 机器可读 JSON Schema |

前端功能变更优先同步 `frontend-feature-logic.md`；涉及架构、权限或数据归属变化时，再同步 `technical-architecture.md`。
